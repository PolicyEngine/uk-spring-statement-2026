# UK Spring Statement 2026

Interactive tool analysing the household impact of the UK Spring Statement 2026, built by [PolicyEngine](https://policyengine.org).

**Live site:** https://uk-spring-statement-2026-policy-engine.vercel.app/

## Features

- **Personal calculator** - Enter your household details to see how the Spring Statement affects your net income
- **Population impact** - Distributional analysis across income deciles, winners and losers, and inequality metrics
- **Economic forecasts** - OBR forecast comparisons before and after the Spring Statement

## Data & methodology

Built using the [Enhanced Family Resources Survey](https://policyengine.org/uk/research/efrs) and the [PolicyEngine UK](https://policyengine.org/uk) microsimulation model. The analysis compares OBR economic forecasts from before and after the Spring Statement to estimate household-level impacts on taxes and benefits.
